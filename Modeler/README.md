# Modeler

## Build

After cloning this repository and setting up the submodules, type the following.

```sh
mkdir build
cd build
cmake ..
cmake --build .
```

## For Mac
```
brew install --cask xquartz
brew install jpeg
brew install libpng
```


Then `Modeler` and `Robotarm` is created under the `build/` directory.
To run it, type `./Modeler` or `Robotarm`.
